# Urban Oasis

App para encontrar refugios del calor en Sevilla: fuentes de agua potable, zonas de sombra y edificios con aire acondicionado.

## Características

- Mapa interactivo con filtros por tipo de oasis
- Geolocalización en tiempo real
- Clima actual (temperatura y viento)
- Diseño mobile-first con Tailwind CSS
- Datos de OpenStreetMap

## Estructura

```
urban-oasis/
├── frontend/          # Angular 21 + Leaflet

```
